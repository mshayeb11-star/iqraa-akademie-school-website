IQRAA Akademie Backend
======================

This backend enables:
- registration form submission
- PDF generation
- email to the academy
- confirmation email to the parent

Setup:
1. Install Node.js
2. Open terminal inside backend folder
3. Run: npm install
4. Copy .env.example to .env
5. Fill in the real SMTP password
6. Run: npm start

Then the frontend registration page can send to:
http://localhost:3000/api/register
